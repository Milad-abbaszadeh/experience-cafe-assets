# Experience Cafe — public assets

Static images for the Experience Cafe website, served via raw.githubusercontent.com
for use in transactional emails (where the recipient's mail client needs a public URL).

- `logo-original.png` — **canonical primary logo**, dark `#1F1F1F` background, 838×456. Use this everywhere the surrounding background isn't already controlled (browser tabs, Knowledge Panel, search-result favicon, social share cards).
- `logo.png` — transparent variant of the same boat. Kept for the existing email template (`EMAIL_LOGO_URL`) and any future use against known-light backgrounds.
